# Deep Co-Planning Workspace

This directory is the live workspace for one Deep Co-Planning run.

## Working Intent

- define or refine the HLD with the human
- pressure-test it through reviewer rounds
- derive one or more implementation-driving specs
- leave bounded known unknowns instead of hidden open questions

## Primary Task Artifacts

- `HLD.md`
- `specs/`

## Round Directories

Each round directory captures one critique-and-synthesis cycle over the current planning state.

Create additional rounds only when the human explicitly asks for another round.
