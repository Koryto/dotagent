# Deep Co-Planning

`deep-co-planning` is an experimental playbook package for phase-level design work that benefits from structured reviewer pressure.

It packages a lead-centric multi-agent planning procedure that starts from a human-led HLD and yields one or more implementation-driving specs.

Use it when:

- the work is too broad for a normal task-level implementation plan
- the human wants reviewer pressure before locking a phase-level design
- the output should later drive multiple tasks or implementation plans

For actual use, start with [START_HERE.md](START_HERE.md).
