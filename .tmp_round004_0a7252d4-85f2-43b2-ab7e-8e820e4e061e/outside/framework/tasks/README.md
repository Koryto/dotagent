# Tasks Namespace

`tasks/` stores task-local working memory.

Use this namespace for:

- implementation plans
- task-specific design notes
- self-review artifacts
- verification notes
- summaries

Recommended naming:

- `{task_name}_ip.md`
- `{task_name}_review.md`
- `{task_name}_verification.md`
- `{task_name}_pr_summary.md`

Example:

- `combat_ability_ip.md`
- `combat_ability_pr_summary.md`

Do not treat task files as durable system truth.
